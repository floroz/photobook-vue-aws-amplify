# photobook-app

Photobook APP in Vue 3, Tailwind, GraphQL and AWS Amplify

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
